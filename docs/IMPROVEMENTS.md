## 三、开发工具链建设

### 3.1 环境初始化脚本

**文件**：`scripts/init-dev.sh`

**功能**：
- 一键初始化开发环境
- 检查依赖（Node.js、pnpm、PostgreSQL）
- 安装项目依赖
- 配置环境变量
- 创建数据库
- 运行数据库迁移
- 填充测试数据

**使用**：
```bash

# 或
./scripts/init-dev.sh
```

**特点**：
- 交互式提示
- 彩色输出
- 错误处理
- 可跳过步骤

### 3.2 数据库备份恢复脚本

**文件**：`scripts/db-backup.sh`

**功能**：
- 备份数据库到 `backups/` 目录
- 恢复数据库
- 列出所有备份
- 清理旧备份（30 天前）

**使用**：
```bash
pnpm db:backup          # 备份
pnpm db:restore         # 恢复
./scripts/db-backup.sh list   # 列出
./scripts/db-backup.sh clean  # 清理
```

**特点**：
- gzip 压缩
- 时间戳命名
- 交互式选择
- 安全确认

### 3.3 代码质量检查脚本

## 四、技术亮点

### 4.1 测试策略

**分层测试**：
- 单元测试：快速反馈，覆盖核心逻辑
- 集成测试：验证模块协作
- E2E 测试：保证用户体验

**Mock 策略**：
- 数据库操作使用 Mock
- 外部 API 调用使用 Mock
- 时间相关使用 Mock

**覆盖率驱动**：
- 设定明确的覆盖率目标
- 持续监控覆盖率变化
- 重点覆盖核心业务逻辑

### 4.2 安全设计

**兑换码安全**：
- HMAC-SHA256 签名验证
- Base32 编码（避免歧义字符）
- 时序安全比较（防止时序攻击）
- 多次使用限制
- 过期时间控制

**密码安全**：
- bcrypt 加密（10 轮）
- 密码强度验证
- 防止暴力破解

**JWT 安全**：
- 7 天有效期
- 签名验证
- 自动刷新机制

### 4.3 点数系统

**双层设计**：
- 每日免费点数（自动重置）
- 购买/兑换点数（永久有效）

**智能消费**：
- 优先消耗免费点数
- 不足时使用购买点数
- 完整的消费记录

**VIP 特权**：
- 免费用户：3 次/天
- VIP 用户：13 次/天
- 多版本生成（VIP 专属）

### 4.4 开发体验

**一键启动**：
```bash
pnpm setup  # 初始化环境
pnpm dev    # 启动开发
```

**便捷工具**：
- 数据库管理（备份、恢复、种子数据）
- 用户管理（创建、授权）
- 兑换码管理（生成、查询）
- 统计查看（用户、订单、生成）

**质量保证**：
- 自动化测试
- 代码规范检查
- 类型检查
- 安全审计
